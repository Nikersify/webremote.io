# webremote.io

### dependencies

#### node.js

get it [here](https://nodejs.org/)

#### bower

```sh
$ npm install -g bower bower-installer
```

#### gulp
```sh
$ npm install -g gulp
```

### running

#### clone the repo

```sh
$ git clone https://github.com/Nikersify/webremote.io.git
$ cd webremote.io/
```

#### get node stuff
*yes, it takes a long time*

```sh
$ npm install
```

#### bower stuff

```sh
$ bower-installer
```

#### run

```sh
$ npm start
```

### building

```sh
$ gulp build
```
builds to `./build/webremote.io/<platform>/`
